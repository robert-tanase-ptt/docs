## Key Components

<AccordionGroup>
  <Accordion title="Assets & Targets">
    Assets represent the systems you're authorized to test. They can be domains, IP addresses, URLs, or network ranges.
  </Accordion>
  <Accordion title="Workspaces">
    Workspaces help you organize assets and scans by project, client, or any other logical grouping.
  </Accordion>
  <Accordion title="Scans">
    Scans are the actual security tests you run against your targets using the platform's tools.
  </Accordion>
  <Accordion title="Findings">
    Findings are the vulnerabilities and security issues discovered during scans.
  </Accordion>
</AccordionGroup>
