## Accessing Settings

Navigate to your profile settings:
1. Click your avatar/name in the top right
2. Select "Settings" or "Profile"

## Profile Information

### Personal Details

| Field | Description |
|-------|-------------|
| Name | Your display name |
| Email | Account email (login) |
| Avatar | Profile picture |
| Company | Organization name |
| Role | Your job title |

## Preferences

### Display Settings

| Setting | Options |
|---------|---------|
| Theme | Light, Dark, System |
| Language | English (more coming) |
| Date Format | MM/DD/YYYY, DD/MM/YYYY |
| Time Zone | Your local timezone |

### Notification Preferences

Configure how you receive notifications:
- Email notification settings
- In-app notification preferences
- Digest frequency

<Tip>
Set your time zone correctly to ensure scheduled scans run at the expected times.
</Tip>

## Default Settings

### Scan Defaults

Set default options for new scans:
- Default scan depth
- Default notification settings
- Default workspace

### Report Preferences

Configure report defaults:
- Default template
- Branding settings
- Export format preference

## API Keys

Manage your API access:

| Action | Description |
|--------|-------------|
| Create | Generate new API key |
| View | See existing keys (masked) |
| Revoke | Disable an API key |

<Warning>
API keys provide full account access. Keep them secure and rotate regularly.
</Warning>
