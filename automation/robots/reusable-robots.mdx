## What Are Reusable Robots?

Reusable robots are templates that can be applied across multiple assets or shared with team members.

## Benefits of Reusable Robots

  <Card title="Standardization" icon="ruler">

  <Card title="Time Saving" icon="clock">
    Build once, use many times.

  <Card title="Team Sharing" icon="users">

  </Card>
  <Card title="Easy Updates" icon="arrows-rotate">

## Creating Reusable Robots

<Steps>
  <Step title="Build Robot">
    Create a robot with your desired workflow.
  </Step>
  <Step title="Use Variables">
    Replace specific values with variables.
  </Step>
  <Step title="Mark as Template">
    Save as a reusable template.
  </Step>
  <Step title="Document">
    Add description and usage instructions.
  </Step>
</Steps>

## Using Variables

Variables make robots flexible:

| Variable Type | Example | Purpose |
|---------------|---------|---------|
| Asset | `{{target_asset}}` | Specify target at runtime |
| Config | `{{scan_depth}}` | Adjust settings per use |
| Output | `{{notification_email}}` | Customize notifications |

## Template Categories

### Assessment Templates

Standard security assessment workflows:
- Initial reconnaissance
- Full vulnerability assessment
- Compliance scanning

### Monitoring Templates

Ongoing security monitoring:
- Weekly scans
- Change detection
- Alert on new findings

### Response Templates

React to findings:
- Severity-based routing
- Auto-ticket creation
- Escalation workflows

## Sharing Robots

### Within Your Team

Team members can:
- View shared templates
- Create instances from templates
- Suggest improvements
