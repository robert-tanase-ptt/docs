The WordPress Scanner performs targeted security testing specifically designed for WordPress sites, checking core, themes, and plugins.

## Checks Performed

### Core WordPress

- Version detection
- Known CVEs for detected version
- Outdated installation warnings
- Debug mode detection

### Plugin Analysis

| Check | Description |
|-------|-------------|
| Enumeration | List installed plugins |
| Version Detection | Identify plugin versions |
| Vulnerability Matching | Check against vuln database |
| Abandoned Plugins | Identify unmaintained plugins |

### Theme Analysis

- Theme detection and versioning
- Known theme vulnerabilities
- Child theme analysis

### Configuration

- User enumeration
- XML-RPC enabled
- wp-config.php exposure
- Directory listing
- Backup files

## Common Findings

| Finding | Risk |
|---------|------|
| Outdated WordPress | High |
| Vulnerable plugins | Variable |
| XML-RPC enabled | Medium |
| User enumeration | Low |
| Debug mode enabled | Medium |

<Warning>
WordPress plugins are a major source of vulnerabilities. Always keep plugins updated and remove unused ones.
</Warning>
