## Overview

The Drupal Scanner performs targeted security assessments of Drupal CMS installations, identifying core, module, and configuration vulnerabilities.

## Checks Performed

### Core Analysis

- Drupal version detection
- Known CVE matching
- Security advisory checks
- Update status

### Module Analysis

| Check | Description |
|-------|-------------|
| Module enumeration | List enabled modules |
| Version detection | Identify module versions |
| Vulnerability check | Match against vuln database |
| Contrib vs Core | Differentiate module types |

### Configuration Checks

- User registration settings
- Error message exposure
- Private file handling
- PHP execution in uploads
- Update module status

## Notable Drupal Vulnerabilities

| CVE | Name | Impact |
|-----|------|--------|
| CVE-2018-7600 | Drupalgeddon 2 | RCE |
| CVE-2019-6340 | REST RCE | RCE |
| Various | SA-CORE advisories | Variable |

<Warning>
Drupal has had several critical RCE vulnerabilities. Keeping Drupal core updated is essential.
</Warning>
