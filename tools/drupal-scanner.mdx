## Overview

The Drupal Scanner identifies vulnerabilities in Drupal installations, including core, modules, and themes. It detects security issues specific to the Drupal CMS.

## Checks performed

### Core checks

| Check | Description |
|-------|-------------|
| **Version detection** | Identify Drupal version |
| **Core vulnerabilities** | Check for known Drupal CVEs (including Drupalgeddon) |
| **CHANGELOG.txt** | Detect exposed changelog |
| **Update status** | Check if updates are available |

### Module checks

| Check | Description |
|-------|-------------|
| **Module enumeration** | Discover installed modules |
| **Version detection** | Identify module versions |
| **Vulnerability matching** | Check against security advisories |

### Configuration checks

| Check | Description |
|-------|-------------|
| **User enumeration** | Discover Drupal users |
| **Admin paths** | Check for exposed admin pages |
| **Backup files** | Check for exposed backups |
| **Error messages** | Check for verbose errors |

## Notable Drupal vulnerabilities

| Vulnerability | CVE | Impact |
|---------------|-----|--------|
| **Drupalgeddon** | CVE-2014-3704 | SQL injection, RCE |
| **Drupalgeddon 2** | CVE-2018-7600 | Remote code execution |
| **Drupalgeddon 3** | CVE-2018-7602 | Remote code execution |

<Warning>
Drupalgeddon vulnerabilities allow complete site takeover. Affected versions must be updated immediately.
</Warning>

## Results

For each finding:
- **Component** — Core, module, or theme name
- **Version** — Detected version
- **Vulnerability** — CVE and description
- **Severity** — Risk rating
- **Advisory** — Drupal security advisory link
