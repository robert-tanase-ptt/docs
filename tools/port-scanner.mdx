## Overview

The Port Scanner identifies open TCP ports on target hosts, revealing running services that may be vulnerable to attack.

## Port Range Options

| Option | Ports | Use Case |
|--------|-------|----------|
| Top 100 | Most common services | Quick discovery |
| Top 1000 | Extended common | Standard assessment |
| Full Scan | All 65,535 | Comprehensive audit |
| Custom | User-defined | Specific requirements |

## Common Ports

| Port | Service | Security Concern |
|------|---------|------------------|
| 21 | FTP | Anonymous access, clear text |
| 22 | SSH | Brute force, outdated versions |
| 23 | Telnet | Clear text transmission |
| 80 | HTTP | Web vulnerabilities |
| 443 | HTTPS | SSL/TLS issues |
| 3306 | MySQL | Database exposure |
| 3389 | RDP | Remote access risks |

## Results Include

For each open port:
- Port number
- Protocol (TCP)
- State (open/filtered)
- Service name
- Version (if detected)
- Banner
