## Overview

The URL Fuzzer discovers hidden or unlinked content on web servers by testing common file and directory names.

## Common Discoveries

| Path | Description |
|------|-------------|
| `/admin` | Admin interfaces |
| `/backup` | Backup files |
| `/config` | Configuration files |
| `/api` | API endpoints |
| `/.git` | Git repository exposure |
| `/phpinfo.php` | PHP configuration |

## Configuration Options

### Extensions

Test for specific file types:
```
.php, .asp, .aspx, .jsp, .html, .bak, .old, .txt
```

### Recursion

- **Off**: Scan base path only
- **Level 1**: Scan discovered directories
- **Level 2+**: Deep recursive scanning

<Warning>
Deep recursive scanning can generate significant traffic. Use thoughtfully in production environments.
</Warning>

## Filtering Results

Filter by response code:
- **200**: Success (found)
- **301/302**: Redirects
- **403**: Forbidden (exists but restricted)
- **404**: Not found (ignore)
