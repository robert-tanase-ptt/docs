## Overview

The Joomla Scanner performs security assessments of Joomla CMS installations, identifying core vulnerabilities, extension issues, and misconfigurations.

## Checks Performed

### Core Analysis

- Version detection
- Known vulnerability matching
- Security patches status
- Update availability

### Extension Analysis

| Type | Checks |
|------|--------|
| Components | Enumeration, vulnerabilities |
| Modules | Detection, security issues |
| Plugins | Analysis, known CVEs |

### Configuration Checks

- Administrator path exposure
- User registration settings
- Error reporting level
- Debug mode status
- Backup files

## Common Findings

| Finding | Severity | Description |
|---------|----------|-------------|
| Outdated core | High | Vulnerable Joomla version |
| Vulnerable extensions | Variable | Third-party security issues |
| Admin path default | Low | /administrator accessible |
| Debug enabled | Medium | Information disclosure |

<Note>
Joomla extensions (especially third-party) are frequently sources of vulnerabilities. Audit and update them regularly.
</Note>

## Results Include

- Joomla version information
- Core vulnerability assessment
- Extension inventory and security status
- Configuration recommendations
