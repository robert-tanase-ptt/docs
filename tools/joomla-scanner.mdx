## Overview

The Joomla Scanner identifies vulnerabilities in Joomla installations, including core, extensions, and templates. It detects security issues specific to the Joomla CMS.

## Checks performed

### Core checks

| Check | Description |
|-------|-------------|
| **Version detection** | Identify Joomla version |
| **Core vulnerabilities** | Check for known Joomla CVEs |
| **README.txt** | Detect exposed readme |
| **Admin path** | Locate administrator directory |

### Extension checks

| Check | Description |
|-------|-------------|
| **Component enumeration** | Discover installed components |
| **Module enumeration** | Discover installed modules |
| **Plugin enumeration** | Discover installed plugins |
| **Vulnerability matching** | Check against security database |

### Configuration checks

| Check | Description |
|-------|-------------|
| **User enumeration** | Discover Joomla users |
| **Debug mode** | Check for enabled debugging |
| **Configuration.php** | Check for exposed configuration |
| **Backup files** | Check for exposed backups |

## Common findings

| Finding | Risk | Recommendation |
|---------|------|----------------|
| Outdated Joomla | High | Update to latest version |
| Vulnerable extension | Varies | Update or remove extension |
| Exposed admin path | Low | Restrict access to /administrator |
| Debug mode enabled | Medium | Disable in production |
| Configuration exposure | Critical | Protect configuration.php |

<Tip>
Regularly update Joomla core and all extensions. Many attacks target known vulnerabilities in outdated installations.
</Tip>

## Results

For each finding:
- **Component** — Core, extension, or template name
- **Version** — Detected version
- **Vulnerability** — CVE and description
- **Severity** — Risk rating
- **References** — Joomla security announcements
