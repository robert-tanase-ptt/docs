## Overview

Website Recon performs reconnaissance on web applications, identifying technologies, frameworks, and configuration details.

## Information Gathered

### Technology Detection

| Category | Examples |
|----------|----------|
| CMS | WordPress, Drupal, Joomla |
| Frameworks | React, Angular, Laravel |
| Web Servers | nginx, Apache, IIS |
| Languages | PHP, Python, Java |
| JavaScript | jQuery, Bootstrap |

### Security Analysis

- HTTP security headers
- Cookie attributes
- SSL/TLS configuration
- Content Security Policy

### Meta Information

- Page titles and descriptions
- robots.txt content
- sitemap.xml
- Interesting paths

## Why Recon Matters

Technology fingerprinting helps:
- Select appropriate scanners
- Identify version-specific vulnerabilities
- Understand the application architecture
- Plan targeted testing

<Tip>
Use Website Recon results to choose which specialized scanners to run (WordPress Scanner for WordPress sites, etc.)
</Tip>
