## Overview

The WAF Detector identifies whether a target website is protected by a Web Application Firewall and attempts to determine which WAF product is in use.

## Why Detect WAFs?

<CardGroup cols={2}>
  <Card title="Testing Strategy" icon="chess">
    Adjust scanning approach based on WAF presence.
  </Card>
  <Card title="Bypass Planning" icon="shield">
    Understand protection mechanisms.
  </Card>
  <Card title="False Positive Reduction" icon="filter">
    Distinguish WAF blocks from real issues.
  </Card>
</CardGroup>

## What It Detects

### WAF Products

Common WAFs detected:
- Cloudflare
- AWS WAF
- Akamai
- Imperva/Incapsula
- F5 BIG-IP ASM
- ModSecurity
- Sucuri

### Detection Methods

| Method | Description |
|--------|-------------|
| Response Headers | WAF-specific headers |
| Cookie Analysis | WAF session cookies |
| Error Responses | Characteristic block pages |
| Behavioral Analysis | Response to test payloads |

## Results Include

- WAF detected (yes/no)
- WAF product name
- Confidence level
- Detection method
- Additional observations
