## Overview

The WHOIS tool retrieves registration information for domains and IP addresses, providing ownership and administrative contact details.

## Domain WHOIS Data

| Field | Description |
|-------|-------------|
| Registrar | Company that registered the domain |
| Created | Domain registration date |
| Expires | Domain expiration date |
| Updated | Last modification date |
| Name Servers | DNS servers |
| Status | Domain status codes |

## IP WHOIS Data

| Field | Description |
|-------|-------------|
| Network | IP range/CIDR |
| Organization | IP block owner |
| Country | Geographic location |
| Abuse Contact | Security contact |

## Privacy Considerations

<Note>
Many domain registrations use privacy protection services that hide actual registrant details. This is normal and expected.
</Note>

## WHOIS for Security

Information useful for penetration testing:
- Organization name for social engineering research
- Email patterns for username guessing
- IP ranges that might be in scope
- Associated domains
