## Overview

The SSL/TLS Scanner analyzes the security of SSL/TLS configurations, identifying weak ciphers, protocol vulnerabilities, and certificate issues.

## Checks performed

### Protocol checks

| Protocol | Status | Notes |
|----------|--------|-------|
| SSL 2.0 | ❌ Deprecated | Broken, never use |
| SSL 3.0 | ❌ Deprecated | POODLE vulnerability |
| TLS 1.0 | ⚠️ Legacy | PCI DSS non-compliant |
| TLS 1.1 | ⚠️ Legacy | Being deprecated |
| TLS 1.2 | ✅ Current | Recommended minimum |
| TLS 1.3 | ✅ Modern | Best security |

### Cipher suite analysis

| Category | Examples |
|----------|----------|
| **Strong** | AES-256-GCM, ChaCha20-Poly1305 |
| **Acceptable** | AES-128-GCM |
| **Weak** | 3DES, RC4 |
| **Insecure** | NULL, EXPORT, DES |

### Certificate checks

| Check | Description |
|-------|-------------|
| **Validity period** | Not expired, not future-dated |
| **Common name** | Matches requested hostname |
| **SANs** | Subject Alternative Names |
| **Chain** | Complete and trusted |
| **Signature algorithm** | SHA-256 or better |
| **Key size** | RSA 2048+, ECDSA 256+ |

### Vulnerability checks

| Vulnerability | CVE | Description |
|---------------|-----|-------------|
| **BEAST** | CVE-2011-3389 | CBC cipher attack |
| **POODLE** | CVE-2014-3566 | SSL 3.0 padding attack |
| **FREAK** | CVE-2015-0204 | Export cipher downgrade |
| **Logjam** | CVE-2015-4000 | DHE export downgrade |
| **DROWN** | CVE-2016-0800 | SSLv2 cross-protocol attack |
| **Heartbleed** | CVE-2014-0160 | OpenSSL memory disclosure |
| **ROBOT** | CVE-2017-13099 | RSA padding oracle |

## Results

For each target:
- **Grade** — Overall security rating (A-F)
- **Protocols** — Supported SSL/TLS versions
- **Cipher suites** — List with strength ratings
- **Certificate details** — Full certificate information
- **Vulnerabilities** — Known issues detected

## Grading

| Grade | Meaning |
|-------|---------|
| **A+** | Excellent configuration, HSTS |
| **A** | Strong configuration |
| **B** | Minor issues |
| **C** | Weak ciphers or protocols |
| **D** | Significant vulnerabilities |
| **F** | Critical issues |

<Tip>
Aim for at least a B grade. A+ requires TLS 1.2+, strong ciphers, and HSTS enabled.
</Tip>
