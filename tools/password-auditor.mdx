## Overview

The Password Auditor tests network services for weak, default, and commonly used credentials that could allow unauthorized access.

## Important Considerations

<Warning>
Password auditing can trigger account lockouts. Coordinate with system administrators and consider using appropriate rate limiting.
</Warning>

### Rate Limiting

- Adjust attempts per second
- Add delays between attempts
- Respect lockout thresholds

### Legal Considerations

- Only test authorized systems
- Document permission
- Follow engagement rules

## Results Include

- Successful credentials found
- Service and port
- Username/password pairs
- Lockout observations

<Tip>
Always change default credentials on all systems, especially network devices and databases.
</Tip>
