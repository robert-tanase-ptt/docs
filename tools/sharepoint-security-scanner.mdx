## Overview

The SharePoint Security Scanner assesses Microsoft SharePoint installations for vulnerabilities, misconfigurations, and security issues.

## Checks Performed

### Version Analysis

| Check | Description |
|-------|-------------|
| Version detection | SharePoint version identification |
| Patch level | Service pack and updates |
| CVE matching | Known vulnerability check |

### Security Checks

- Anonymous access settings
- Web part vulnerabilities
- Document library exposure
- Search disclosure
- User enumeration

### Authentication

- Login page analysis
- Authentication bypass tests
- Session management
- Brute force protection

## SharePoint-Specific Risks

| Risk | Impact |
|------|--------|
| CVE-2019-0604 | Remote Code Execution |
| Anonymous Access | Data exposure |
| Misconfigurations | Information disclosure |

<Warning>
SharePoint often contains sensitive corporate data. Vulnerabilities can lead to significant data breaches.
</Warning>
