## Overview

The SharePoint Security Scanner identifies vulnerabilities and misconfigurations in Microsoft SharePoint deployments, both on-premises and SharePoint Online.

## Checks performed

### Version checks

| Check | Description |
|-------|-------------|
| **Version detection** | Identify SharePoint version |
| **Patch level** | Determine installed updates |
| **CVE matching** | Check for known vulnerabilities |

### Security checks

| Check | Description |
|-------|-------------|
| **Anonymous access** | Check for unauthenticated access |
| **Directory browsing** | Check for enabled browsing |
| **Exposed lists** | Find accessible lists |
| **Document libraries** | Check for exposed documents |

### Authentication checks

| Check | Description |
|-------|-------------|
| **Authentication methods** | Identify enabled auth types |
| **Forms authentication** | Check for form-based auth issues |
| **Kerberos/NTLM** | Test Windows authentication |

## SharePoint-specific risks

| Risk | Description |
|------|-------------|
| **CVE-2019-0604** | Remote code execution vulnerability |
| **CVE-2020-1147** | XML deserialization RCE |
| **Overly permissive sharing** | Documents shared with "Everyone" |
| **Anonymous access** | Sites accessible without login |

<Warning>
SharePoint vulnerabilities can expose sensitive corporate documents. Regular security assessments are critical.
</Warning>

## Results

For each finding:
- **Issue type** — Vulnerability or misconfiguration
- **Location** — Affected site/list/library
- **Severity** — Risk rating
- **Details** — Technical description
- **Remediation** — Steps to fix
