## What Are Report Templates?

Report templates define the structure, styling, and content of generated reports. They ensure consistency across your deliverables.

## Template Components

### Header and Branding

Customize report appearance:
- Company logo
- Report title format
- Cover page design
- Color scheme

### Section Configuration

Control which sections appear:
- Executive summary
- Scope and methodology
- Detailed findings
- Risk ratings
- Remediation recommendations
- Technical appendices

### Finding Details

Configure how findings are presented:

| Option | Description |
|--------|-------------|
| Severity display | How risk levels are shown |
| Evidence inclusion | Screenshots, requests, responses |
| Remediation depth | Brief or detailed guidance |
| Reference links | CVE, CWE, OWASP links |
