## The Scan Page

The scan page is your central hub for managing and monitoring security scans. Here you can launch new scans, track progress, and access results.

## Page Layout

<CardGroup cols={2}>
  <Card title="Scan List" icon="list">
    View all your scans with status, target, and timestamp information.
  </Card>
  <Card title="Quick Actions" icon="bolt">
    Start new scans or re-run previous configurations.
  </Card>
  <Card title="Filters" icon="filter">
    Filter scans by status, tool, target, or date range.
  </Card>
  <Card title="Results Summary" icon="chart-bar">
    See findings count and severity breakdown at a glance.
  </Card>
</CardGroup>

## Scan List Columns

| Column | Description |
|--------|-------------|
| Status | Current state of the scan (Queued, Running, Completed, etc.) |
| Tool | Which security tool was used |
| Target | The asset being scanned |
| Findings | Number of issues discovered |
| Started | When the scan began |
| Duration | How long the scan took |

## Available Actions

From the scan page, you can:

- **Start New Scan**: Launch a fresh scan with custom settings
- **Re-run Scan**: Repeat a previous scan with the same configuration
- **View Results**: Access detailed findings and evidence
- **Export**: Download scan results in various formats
- **Delete**: Remove scans you no longer need

## Scan Details View

Clicking on a scan opens the detailed view showing:

- Complete scan configuration
- All findings with severity levels
- Raw output and evidence
- Scan logs and timeline

<Tip>
Use the filters to quickly find scans by status or tool type. This is especially helpful when managing many concurrent scans.
</Tip>
