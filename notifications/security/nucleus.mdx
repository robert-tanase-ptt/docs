## Overview

The Nucleus integration enables you to send security findings to Nucleus for centralized vulnerability management and tracking.

## What Is Nucleus?

Nucleus is a vulnerability management platform that aggregates findings from multiple security tools, providing a unified view of your security posture.

## Setting Up Nucleus

<Steps>
  <Step title="Get API Credentials">
    In Nucleus, obtain your API key and connector ID.
  </Step>
  <Step title="Configure Integration">
    In Pentest-Tools.com, go to Settings → Integrations → Nucleus.
  </Step>
  <Step title="Enter Credentials">
    Add your Nucleus API credentials.
  </Step>
  <Step title="Test Connection">
    Verify the integration works.
  </Step>
</Steps>

## Data Sent to Nucleus

| Data | Format |
|------|--------|
| Findings | Nucleus-compatible format |
| Asset Info | IP, hostname, URL |
| Severity | Mapped to Nucleus levels |
| Evidence | References and proof |

## Sync Behavior

### What Syncs

- New findings from scans
- Finding updates and status changes
- Asset information
- Remediation guidance

### Sync Options

| Option | Description |
|--------|-------------|
| Real-time | Immediate sync on scan complete |
| Batch | Periodic batch uploads |
| Manual | On-demand sync |

## Mapping Configuration

Map Pentest-Tools.com data to Nucleus fields:

| Platform | Nucleus |
|----------|---------|
| Critical | Critical |
| High | High |
| Medium | Medium |
| Low | Low |
| Info | Informational |

<Tip>
Use consistent severity mapping across all tools feeding into Nucleus for accurate prioritization.
</Tip>
