## Choosing the right plan

<Tabs>
  <Tab title="NetSec">

    - Deep network vulnerability scanning
    - Cloud scanner for AWS, Azure, GCP misconfigurations

    - Password auditing
    - API access for automation
    - Team collaboration features
    - Most third-party integrations
  </Tab>
  <Tab title="WebNetSec">

    **Ideal use cases**:

    - Full-stack vulnerability assessment
    - CI/CD pipeline integration

    **Key capabilities**:
    - Everything in NetSec, plus:
    - Deep web vulnerability scanning
    - Authenticated web application scanning
    - API vulnerability scanning

## Free tier limitations

The free tier allows you to explore the platform with significant restrictions:

- **Light scan presets only** – No access to deep scanning capabilities
- **No API access** – Manual scans through the web interface only
- **No automation** – No pentest robots or scheduled scans
- **No team features** – Single user access only
- **Limited integrations** – Email notifications only
- **No VPN scanning** – External targets only

<Tip>
Start with a 14-day free trial of any paid plan to experience the full capabilities before committing to a subscription.
</Tip>

## Billing cycles

All paid plans support two billing options:

| Billing cycle | Description |
|---------------|-------------|
| Monthly | Pay month-to-month with flexibility to change or cancel anytime |
| Annual | Pay yearly for a discounted rate |

## Upgrading your plan

<Steps>
  <Step title="Compare features">
    Review the [Feature matrix](/plans-and-limits/feature-matrix) to understand what each plan offers.
  </Step>
  <Step title="Access billing settings">
    Navigate to **Settings → Billing → Change plan** in your account.
  </Step>
  <Step title="Select your new plan">
    Choose between NetSec, WebNetSec, or Pentest Suite, and select monthly or annual billing.
  </Step>
  <Step title="Configure assets">
    Select the number of assets you need for your scanning requirements.
  </Step>
  <Step title="Complete checkout">
    Review your order and complete payment through FastSpring, our secure payment processor.
  </Step>
</Steps>

<Note>
When upgrading mid-cycle, you'll be charged a prorated amount for the remainder of your current billing period.
</Note>
